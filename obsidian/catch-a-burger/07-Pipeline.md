
**ФАЗА 0 — Фундамент (Дни 1–3).** Цель: чистый каркас, на котором всё стоит.

- **Day 1** — правильный Bootstrap: ожидание готовности SDK (`onGetSDKData`) → чтение сейвов → переход в Menu. Убрать `async void`, убрать тестовые `TestSignal`/`DebugEventLogger` в песочницу. Развести ProjectContext (глобальное) и SceneContext.
- **Day 2** — ProjectInstaller: обёртка над Plugin YG (`YandexSDKWrapper`), система сохранений (заглушка), каркас AudioManager. Глобальные сигналы объявлены в проектном контейнере.
- **Day 3** — переходы между сценами Bootstrap→Menu→Gameplay→Menu, GameFlowController-скелет с пустым автоматом состояний (Booting/Ready/Running/Paused/Resuming/GameOver — пока пустые).
- **🎯 Веха:** игра запускается, ждёт SDK, ходит по сценам, не падает. Геймплея нет.

**ФАЗА 1 — Core loop на квадратах (Дни 4–14). Самая важная фаза.** Цель: играбельное ядро. Серые квадраты, ноль арта.

- **Day 4–5** — ConveyorSystem: две ленты, квадраты-ингредиенты едут, базовая скорость. Игровое время (замораживаемое) vs реальное — заложить сразу.
- **Day 6–7** — ChefController: повар-квадрат, доскок влево/вправо, ввод тапом по половине экрана. Сигналы ChefMoved.
- **Day 8** — ловля: повар на стороне + бездействие = поймал. Сигнал IngredientCaught. Пока без заказа — просто ловит.
- **Day 9** — отбивание сковородой: тап по своей стороне = удар, ингредиент отскакивает (DOTween). Кулдаун удара. Сигнал IngredientHit.
- **Day 10** — IngredientDropped → LivesSystem → −1 жизнь → GameOver. Первый полный риск-цикл.
- **Day 11–12** — OrderSystem: фиксированный заказ, «нужный следующий», совпадение/ошибка. Сигналы OrderItemMatched/Wrong/Completed.
- **Day 13** — ScoringSystem: цена бургера, грязные слои, комбо. Сигнал на OrderCompleted → новый заказ, рост скорости.
- **Day 14** — связать всё: собрал бургер → продал → новый → быстрее. Умер → game over → рестарт.
- **🎯 Веха (КРИТИЧЕСКАЯ):** в игру **можно играть и она затягивает** на серых квадратах. Если на Day 14 не затягивает даже без арта — **стоп, чиним механику, а не едем дальше.** Арт не спасёт скучное ядро.

**ФАЗА 2 — Спавн-директор (Дни 15–21). Сердце честности.**

- **Day 15–16** — бюджет угроз: директор считает «решаемо/нет» перед спавном (T_react, доскок, кулдаун, время полёта).
- **Day 17** — окно безопасности между сторонами, подмешивание мусора.
- **Day 18** — категория «б» (ингредиенты заказа не в свой черёд).
- **Day 19–20** — кривая сложности: сжатие окна со временем. Тут много ручной настройки на ощущениях.
- **Day 21** — пауза по фокусу вкладки (`visibilitychange`/SDK) + Resuming-отсчёт 3-2-1 как единый выход из любой паузы.
- **🎯 Веха:** игра честная (нет нечестных смертей) и сложность растёт ощутимо. Сворачивание вкладки не убивает.

**ФАЗА 3 — Монетизация (Дни 22–28).**

- **Day 22–23** — AdManager: interstitial на game over с частотным лимитом. Обработка всех колбэков YG.
- **Day 24** — rewarded-continue (+1 жизнь), награда строго по `onRewarded`.
- **Day 25** — rewarded-буст в забеге.
- **Day 26–27** — бусты: три штуки (замедление, ×2 цены, чистый заказ), BoostController, эффекты.
- **Day 28** — выбор бустов «с собой» перед забегом.
- **🎯 Веха:** все рекламные события стреляют, бусты работают. Это момент, когда игра начинает «зарабатывать».

**ФАЗА 4 — Мета и экономика (Дни 29–35).**

- **Day 29–30** — Menu: выбор героя (Рома/Ника), кнопка играть, баланс валюты.
- **Day 31** — CurrencySystem: валюта за забеги, сохранение.
- **Day 32–33** — магазин: скины и бусты за валюту/инап. Инап «убрать рекламу».
- **Day 34** — лидерборд Яндекса (рекорд → таблица).
- **Day 35** — сохранения целиком: рекорд, валюта, скины, бусты, флаг no-ads. Облако + локально.
- **🎯 Веха:** полный продуктовый цикл — играю, зарабатываю, трачу, возвращаюсь.

**ФАЗА 5 — Арт и фидбэк (Дни 36–40).**

- **Day 36–37** — спрайты ингредиентов, лент, фона (AI-арт/бесплатные ассеты). Квадраты → бургеры.
- **Day 38** — герои Рома/Ника: пока статика или простая скелетная (2D Animation — ставим здесь, не раньше).
- **Day 39** — game feel: тряска экрана, частицы поимки, отскок, звук (через StreamingAssets — твой YG-воркэраунд).
- **Day 40** — UI-полировка, иконки тикета, анимация цены/комбо.
- **🎯 Веха:** игра выглядит как продукт, а не прототип.

- **Day 43** — фокус-группа: твои дети играют, ловим фрустрацию и баги баланса.

- **🎯 Веха:** завершённый билд на модерацию.
